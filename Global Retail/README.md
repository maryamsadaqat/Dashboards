# Global Retail Dashboard

## Overview

This project is a Power BI dashboard that provides a revenue performance overview for a global retail business. It consolidates sales, profit, and order data into a single interactive report, allowing stakeholders to track key performance indicators and identify trends across time, product categories, and countries.

## Key Metrics

- Total Sales: $12.64M
- Total Profit: $1.47M
- Profit Margin: 11.61%
- Total Orders: 178,312
- Return Rate: 0.61%

## Features

- Year filter (2012 to 2015) for time-based analysis
- Revenue and Profit Margin trend over the year
- Monthly Sales and Profit Comparison
- Revenue breakdown by Quarter and Category (Furniture, Office Supplies, Technology)
- Revenue by Top 5 Countries

## Tools Used

- Power BI Desktop
- DAX for calculated measures
- Power Query for data cleaning and transformation

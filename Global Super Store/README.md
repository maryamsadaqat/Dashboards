# Global Superstore Sales Dashboard

## Overview
This project is a Power BI dashboard that provides a sales and profitability overview for a global superstore business. It consolidates orders, returns, and customer data into a single interactive report, allowing stakeholders to track key performance indicators and identify trends across time, regions, markets, and product categories.

## Key Metrics
- Total Sales: $62.27M
- Total Profit: $1.51M
- Profit Ratio: 2.42%
- Returned Orders: 992 (Asia Pacific, Europe, USCA, LATAM, Africa)

## Features
- Year filter (2012 to 2015) for time-based analysis
- Total Revenue & Profit dynamic trend over the years (toggle buttons to switch metric view)
- Category Sales Analysis by Year (Furniture, Office Supplies, Technology)
- Total Sales breakdown by Category (donut chart)
- Top 3 Regional Managers by Sales
- Region Contribution by Profit Ratio
- Returned Orders by Market (treemap)
- Market slicer for dynamic filtering
- Natural language Q&A page for ad-hoc data exploration

## Tools Used
- Power BI Desktop
- DAX for calculated measures
- Power Query for data cleaning and transformation
